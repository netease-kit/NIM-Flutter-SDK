toType property

QChatSystemMessageToType? toType
final

通知发送对象类型，对应下面四种情况，不需要主动设置

QChatSystemMsgToType.server：如果只填了serverId，则发送给serverId下的所有人 QChatSystemMsgToType.channel: 如果填了serverId+channelId，则发送给serverId+channelId下的所有人 QChatSystemMsgToType.server_ACCIDS: 如果填了serverId+toAccids，则发送给server下的指定账号列表 QChatSystemMsgToType.channel_ACCIDS: 如果填了serverId+channelId+toAccids，则发送给server下某个channel里的指定账号列表
